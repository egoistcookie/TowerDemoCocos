# 项目样式规范

本文档记录了项目中使用的 UI 样式规范，后续开发应参照此规范保持风格一致性。

---

## 按钮样式

### 标准按钮（龙肉选择框按钮）

```typescript
// 按钮背景 + 边框 - 使用同一个 Graphics 组件
const graphics = buttonNode.addComponent(Graphics);

// 先绘制填充
graphics.fillColor = new Color(60, 60, 60, 255);  // 深灰色背景
graphics.roundRect(-width / 2, -height / 2, width, height, 8);
graphics.fill();

// 再绘制边框
graphics.strokeColor = new Color(255, 255, 255, 255);  // 白色边框
graphics.lineWidth = 4;
graphics.roundRect(-width / 2, -height / 2, width, height, 8);
graphics.stroke();

// 按钮文字
const label = labelNode.addComponent(Label);
label.fontSize = 18;  // 关闭按钮为 30
label.color = new Color(255, 255, 255, 255);  // 白色文字
label.horizontalAlign = Label.HorizontalAlign.CENTER;
label.verticalAlign = Label.VerticalAlign.CENTER;
```

**样式参数：**
| 属性 | 值 | 说明 |
|------|-----|------|
| 背景色 | `Color(60, 60, 60, 255)` | 深灰色 |
| 边框色 | `Color(255, 255, 255, 255)` | 白色 |
| 边框宽度 | `4` | 较粗边框 |
| 圆角 | `8` | 适度圆角 |
| 文字大小 | `18` | 标准按钮 / 关闭按钮 `30` |
| 文字颜色 | `Color(255, 255, 255, 255)` | 白色 |

**尺寸参考：**
- 标准按钮：`120 x 50`
- 关闭按钮（×）：`40 x 40`

## 弹窗样式

### 选择框面板（龙肉选择框）

```typescript
const containerNode = new Node('DragonMeatSelectionPanel');
const uiTransform = containerNode.addComponent(UITransform);
uiTransform.setContentSize(500, 355);

const graphics = containerNode.addComponent(Graphics);
// 背景
graphics.fillColor = new Color(0, 0, 0, 220);  // 半透明黑色
graphics.roundRect(-250, -177.5, 500, 355, 15);
graphics.fill();
// 边框
graphics.strokeColor = new Color(255, 215, 0, 255);  // 金色边框
graphics.lineWidth = 4;
graphics.roundRect(-250, -177.5, 500, 355, 15);
graphics.stroke();
```

**样式参数：**
| 属性 | 值 | 说明 |
|------|-----|------|
| 面板尺寸 | `500 x 355` | 龙肉选择框固定尺寸 |
| 背景色 | `Color(0, 0, 0, 220)` | 半透明黑色 |
| 边框色 | `Color(255, 215, 0, 255)` | 金色 |
| 边框宽度 | `4` | 较粗边框 |
| 圆角 | `15` | 大圆角 |

**遮罩层：**
```typescript
const maskLayer = new Node('DragonMeatMask');
```

**布局结构（上下结构）：**

- 第三部分：三个按钮（居中排列），Y = -110
- 关闭按钮：右上角 ×，位置 (220, 145)，尺寸 40x40，字体大小 30

### 单位介绍弹窗（单位首次出现提示框）

```typescript
// 容器节点（UnitIntroPopup.container）
const container = containerNode; // 预制体中配置
const uiTransform = container.addComponent(UITransform);
// 尺寸根据预制体配置

// 背景绘制（Graphics 动态绘制）
const graphics = container.getComponent(Graphics);
graphics.fillColor = new Color(0, 0, 0, 200);
graphics.roundRect(-popupWidth / 2, -popupHeight / 2, popupWidth, popupHeight, 15);
graphics.fill();
graphics.strokeColor = borderColor || new Color(100, 200, 255, 255); // 可选自定义边框色
graphics.lineWidth = borderColor ? 5 : 3;
graphics.roundRect(-popupWidth / 2, -popupHeight / 2, popupWidth, popupHeight, 15);
graphics.stroke();
```

**样式参数：**
| 属性 | 值 | 说明 |
|------|-----|------|
| 面板尺寸 | 预制体配置 | 通常为 `400 x 300` 左右 |
| 背景色 | `Color(0, 0, 0, 200)` | 半透明黑色 |
| 默认边框 | `Color(100, 200, 255, 255)` | 浅蓝色 |
| 自定义边框 | 可选传入 | 如兽人燃血狂暴使用特殊色 |
| 边框宽度 | `3` 或 `5` | 自定义边框时更粗 |
| 圆角 | `15` | 大圆角 |

**遮罩层：**
```typescript
const maskLayer = new Node('UnitIntroMask');
const maskGraphics = maskLayer.addComponent(Graphics);
maskGraphics.fillColor = new Color(0, 0, 0, 180); // 半透明黑色（约 70% 不透明度）
```

**布局结构：**
- 单位图标：居中，Y 轴位置根据内容调整
- 单位名称：居中，字体大小 22，白色
- 单位描述：居中，字体大小 18-22，白色，支持多行
- 关闭按钮：右上角 × 或底部按钮

## 代码模板

### 创建按钮完整示例

```typescript
private createButton(parent: Node, name: string, text: string, pos: Vec3, width: number, height: number, onClick: () => void): Node {
    const buttonNode = new Node(name);
    buttonNode.setParent(parent);
    buttonNode.setPosition(pos);

    const btnTransform = buttonNode.addComponent(UITransform);
    btnTransform.setContentSize(width, height);

    // 按钮背景和边框
    const graphics = buttonNode.addComponent(Graphics);
    graphics.fillColor = new Color(60, 60, 60, 255);
    graphics.roundRect(-width / 2, -height / 2, width, height, 8);
    graphics.fill();
    graphics.strokeColor = new Color(255, 255, 255, 255);
    graphics.lineWidth = 4;
    graphics.roundRect(-width / 2, -height / 2, width, height, 8);
    graphics.stroke();

    // 按钮文字
    const labelNode = new Node('Label');
    labelNode.setParent(buttonNode);
    labelNode.setPosition(0, 0, 0);
    const labelTransform = labelNode.addComponent(UITransform);
    labelTransform.setContentSize(width - 10, height);
    const label = labelNode.addComponent(Label);
    label.string = text;
    label.fontSize = 18;
    label.color = new Color(255, 255, 255, 255);
    label.horizontalAlign = Label.HorizontalAlign.CENTER;
    label.verticalAlign = Label.VerticalAlign.CENTER;

    // 点击事件
    buttonNode.on(Node.EventType.TOUCH_END, onClick, this);

    return buttonNode;
}
```
