**复现 SQL（与脚本 `last7_daily_draw_finish` 同结构，时间窗换为本期起止即可）**

```sql
SELECT DATE(FROM_UNIXTIME(gs.start_time/1000)) AS d,
       COUNT(*) AS total_sessions,
       ROUND(SUM(CASE WHEN EXISTS (
         SELECT 1 FROM card_selection_events c
         WHERE c.session_id = gs.id AND c.game_time != 0
       ) THEN 1 ELSE 0 END) * 100.0 / COUNT(*), 2) AS first_draw_rate_pct,
       ROUND(SUM(CASE WHEN gs.result IS NOT NULL THEN 1 ELSE 0 END) * 100.0 / COUNT(*), 2) AS finish_rate_pct
FROM game_sessions gs
WHERE /* 与清晰口径一致的非测试、非 unknown 渠道 */
  AND gs.start_time BETWEEN UNIX_TIMESTAMP('2026-04-24 00:00:00') * 1000
                        AND UNIX_TIMESTAMP('2026-05-07 23:59:59') * 1000
GROUP BY DATE(FROM_UNIXTIME(gs.start_time/1000))
ORDER BY d;
```

## 七、清晰口径

1. 非测试记录：固定排除测试 player_id，且排除 `visitor_source_records.channel='unknown'`。
2. 至少一抽：`id in (select session_id from card_selection_events where game_time != 0)`。
3. 正常结束：`result is not null`。
4. 广告抽卡：`selection_mode in ('get_all','reroll','single_video')`。
5. 选卡统计：优先读 `select_buff_card.details` 的 `mode|unitId|rarity|buffType`。
6. **6.3 首抽/完赛**：按**日历日**分母为当日**全部**开局会话（含未完成）；与「可比口径」（至少一抽且正常结束）不同，**不可直接对比第二节胜率表**。
