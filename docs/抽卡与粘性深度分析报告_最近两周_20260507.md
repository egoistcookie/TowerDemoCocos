## 八、清晰口径

1. **非测试记录**：固定排除测试 `player_id`，且排除 `visitor_source_records.channel='unknown'`。  
2. **可比口径（第二节主表）**：至少一抽（`game_time != 0`）且正常结束（`result is not null`）。  
3. **粘性弃局**：须 `game_time > 0`；最后一次有效选卡为 `event_time` 最大且 `game_time>0` 的那条记录的局内秒数。  
4. **首抽率（6.3）**：按日历日，存在 `game_time != 0` 的选卡之会话数 ÷ 当日开局会话总数。  
5. **完赛率（6.3）**：按日历日，`result IS NOT NULL` 会话数 ÷ 当日开局会话总数。  
6. **最近 7 日历日**：`meta.last7_calendar_begin` 至本期 `--end`，共 7 个自然日。  
7. **关卡占比**：按 `game_sessions.level`、当日开局为分母。  
8. **广告抽卡**：`selection_mode in ('get_all','reroll','single_video')`。  
9. **选卡统计**：优先读 `select_buff_card.details` 的 `mode|unitId|rarity|buffType`。  
10. **版本标签**：按会话开始日历日映射 `VERSIONS`；同日多次发版无法拆到小时。

### 附录：SQL 摘要（与脚本一致）

**A. 弃局（prev7 / last7）**：见历史附录或脚本 `stickiness_prev7_last7`。

**B. 逐日首抽率、完赛率**

```sql
SELECT DATE(FROM_UNIXTIME(gs.start_time/1000)) AS d,
       ROUND(SUM(CASE WHEN EXISTS (
         SELECT 1 FROM card_selection_events c
         WHERE c.session_id = gs.id AND c.game_time != 0
       ) THEN 1 ELSE 0 END) * 100.0 / COUNT(*), 2) AS first_draw_rate_pct,
       ROUND(SUM(CASE WHEN gs.result IS NOT NULL THEN 1 ELSE 0 END) * 100.0 / COUNT(*), 2) AS finish_rate_pct
FROM game_sessions gs
WHERE /* base_session */
  AND gs.start_time BETWEEN UNIX_TIMESTAMP('last7_calendar_begin') * 1000 AND UNIX_TIMESTAMP('end') * 1000
GROUP BY DATE(FROM_UNIXTIME(gs.start_time/1000))
ORDER BY d;
```

**C. 逐日关卡占比**

```sql
SELECT DATE(FROM_UNIXTIME(gs.start_time/1000)) AS d, gs.level,
       ROUND(COUNT(*) * 100.0 / SUM(COUNT(*)) OVER (
         PARTITION BY DATE(FROM_UNIXTIME(gs.start_time/1000))
       ), 2) AS pct_within_day
FROM game_sessions gs
WHERE /* base_session */
  AND gs.start_time BETWEEN UNIX_TIMESTAMP('last7_calendar_begin') * 1000 AND UNIX_TIMESTAMP('end') * 1000
GROUP BY DATE(FROM_UNIXTIME(gs.start_time/1000)), gs.level
ORDER BY d, gs.level;
```
