**裂缝贴图推荐位置：索引 3（Background 之后，GameManager 之前）**

说明：
- 所有路径均以 `Canvas` 为根进行非递归定位
- Canvas 下节点顺序决定了渲染层级，索引越小越先渲染（在底层）
- 索引 2 的 Background 是场景背景，所有游戏单位应放在其后
- 索引 10-30 左右是主要游戏单位容器
- 索引 33+ 是 UI 层，覆盖在游戏单位之上
