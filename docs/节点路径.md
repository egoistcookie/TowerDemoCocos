# 节点路径（Node Paths）

用于统一约定关键节点的固定路径，便于代码及后续 AI 引用。

- GameManager
  - Canvas/GameManager

- 敌人与容器
  - Canvas/Enemies
  - Canvas/Portals

- 我方单位容器（如存在以下命名）
  - Canvas/Towers          （弓箭手等远程单位）
  - Canvas/Hunters
  - Canvas/ElfSwordsmans   （或 Canvas/Swordsmen）
  - Canvas/Priests
  - Canvas/Mages

- UI / 背景（如存在以下命名）
  - Canvas/UI
  - Canvas/HUD
  - Canvas/TopUI
  - Canvas/Background（或 Canvas/BG / Canvas/Back）

- 天赋页（Talent）节点
  - Canvas/BottomSelection/TalentPanel/TalentUnitScrollView/View/UnitCardsContainer
    - 说明：单位卡片容器（UnitCardsContainer）的固定路径，用于刷新单位等级等信息

说明：
- 所有路径均以 `Canvas` 为根进行非递归定位。
- 若实际场景中存在不同命名，请在此文档中补充/更新，以保持一致性。
